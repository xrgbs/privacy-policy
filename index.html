Privacy Policy

Last Updated: October 21, 2025

Simple Version: This is a personal tool I built for myself to track my own eBay orders. I'm the only user. I only access my own eBay data, store it locally for my own reference, and don't share it with anyone.

1. Who This Application Is For

This application ("Personal eBay Package Manager") is a personal project created and operated solely by me for my own use. I am both the developer and the only user of this application.

2. What Data I Collect

Through the eBay API, this application accesses:

  • My eBay order IDs
  • Order contents and item details
  • Purchase dates and tracking information
  • Seller information
  • Other order-related data provided by the eBay API

3. How I Use This Data

I use this data exclusively for:

  • Personal package tracking and management
  • Organizing my own purchase history
  • Managing my own orders and deliveries

This data is never used for:

  • Marketing or advertising
  • Training AI models or algorithms
  • Analytics or aggregate statistics
  • Any commercial purposes

4. Data Storage and Security

All data is stored locally on my personal devices. I implement reasonable security measures to protect this data, including:

  • Encrypted storage where applicable
  • Secure API credentials management
  • No cloud storage or third-party services

5. Data Sharing

I do not share, sell, rent, or transfer any data to third parties. Ever. This is a personal tool for personal use only.

6. Data Retention and Deletion

I retain this data only as long as it's useful for my personal package management. I will delete data when:

  • It's no longer needed for my purposes
  • Requested by eBay
  • I discontinue use of this application

7. Compliance with eBay's Privacy Notice

This application and my use of eBay data fully complies with eBay's Privacy Notice. I only process data in ways that eBay itself is permitted to process it.

8. Your Rights

Since I'm the only user of this application accessing my own data, I have full control over:

  • Accessing my data at any time
  • Modifying or deleting my data
  • Revoking API access through eBay's user settings
  • Discontinuing use of the application

9. Third-Party Services

This application connects only to eBay's API. I do not use any other third-party services, analytics tools, or tracking mechanisms.

10. Changes to This Policy

If I make changes to how I handle data, I'll update this policy and the "Last Updated" date above.

11. Contact

Since this is a personal project for my own use, if eBay or any relevant authority needs to contact me regarding this application, they can reach me through my eBay Developer account contact information.

12. Legal Compliance

This application complies with:

  • eBay's API License Agreement
  • eBay's Developer Program Terms
  • Applicable data protection laws (GDPR, CCPA, etc.)
